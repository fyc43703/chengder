主要功能說明

📂 匯入、清除、主題

「 📂 匯入PDF 檔案 」: 點擊以選擇您電腦中的 PDF 檔案並載入。
「 🧹 清除 」: 點擊以清除「搜尋文字」和「取代文字」兩個輸入框的內容。
「 🌓 (切換器) 」: 切換淺色或深色（暗黑）操作模式。

🔍 搜尋與取代

「 搜尋文字 」: 輸入您想在 PDF 中尋找的文字。右側 [ 0 ] 會即時顯示找到的數量。
「 取代文字 」: 輸入您要替換成的新文字。
「 依序替換 」: 替換目前（橘色框）反白的文字，並自動跳至下一個。
「 全部替換 」: 一次將文件中所有找到的「搜尋文字」全部替換。

🧭 頁面導覽與縮放

頁面導覽：直接在預覽區塊中上下滾動，即可瀏覽所有頁面。
頁碼顯示：操作列上的指示器會即時顯示您目前所在的頁數。
縮放功能：使用「縮小」、「放大」按鈕調整比例，或點擊「整頁」讓頁面自動縮放至最適寬度。

💾 儲存與匯出

「 💾 儲存... 」: 點擊此按鈕會彈出儲存選項。
「 💾 另存為 PDF 」: 將您修改後的內容（包含替換的文字）儲存為一個新的 PDF 檔案。
「 🖼️ 另存為 PNG (ZIP) 」: 將 PDF 的每一頁（包含修改內容）分別存為 PNG 圖片，並打包成一個 ZIP 檔案下載。
